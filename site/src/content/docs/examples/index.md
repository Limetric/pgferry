The repository ships real example configs instead of pseudo-doc snippets. Use them as starting points, then trim them down for your schema.

## Source groups

- [MySQL examples](/examples/mysql/)
- [SQLite examples](/examples/sqlite/)
- [MSSQL examples](/examples/mssql/)

## Patterns covered

- minimal safe runs
- recreate-fast variants
- hook-driven migrations
- schema-only and data-only runs
- resumable chunked execution
- a full MySQL `sakila` example
